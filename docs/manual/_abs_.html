abs

A function that can be used in Formulas. The absolute value.

Syntax and semantics

abs (x)
compute the absolute value of x.

Examples

writeInfoLine: abs (3.14), " ", abs (0), " ", abs (-789e-300)
 =>
3.14 0 7.89e-298
