upperCaseAppName$

A function that can be used in Formulas. The name of the present app, in upper camel case, such as Praat or PraatForHospitals.

Syntax and semantics

upperCaseAppName$ ( )
give the name of the present app, in upper camel case.

Examples

writeInfoLine: upperCaseAppName$ ( )
 =>
Praat
