part##

A function that can be used in Formulas.

Syntax and semantics

part## (matrix#, fromRow, toRow, fromColumn, toColumn)
compute a subwindow of cells of a matrix.

Examples

assert part## ({{ 7, 4, 9, 3 }, { 1, 4, 3, 22 }, { 0, 8, 6, 5 }},
 ... 2, 3, 1, 3) = {{ 1, 4, 3 }, { 0, 8, 6 }}
