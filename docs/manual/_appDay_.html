appDay

A function that can be used in Formulas. The month day on which the present version of Praat was built (a number between 1 and 31).

Syntax and semantics

appDay ( )
give the month day of the creation of the present version of Praat.

Examples

writeInfoLine: appDay ( )
 =>
25
