Dissimilarity & Configuration: Get stress (monotone mds)...

A command to obtain the stress value for the selected Dissimilarity and Configuration object.

Behaviour

We use stress formulas that are independent of the scale of the Configuration: you would have got the same stress value if you had pre-multiplied the selected Configuration with any number greater than zero.
