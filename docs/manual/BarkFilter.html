BarkFilter

A deprecated type of object in Praat. It has been replaced by BarkSpectrogram.

An object of type BarkFilter represents an acoustic time-frequency representation of a sound: the power spectral density P(z, t), expressed in dB's as 10*log10(power/4e-10)). In the now preferred BarkSpectrogram the power is represented instead of its dB value.It is sampled into a number of points around equally spaced times ti and frequencies zj (on a Bark scale).

Inside a BarkFilter

With Inspect you will see that this type contains the same attributes a Matrix object.
