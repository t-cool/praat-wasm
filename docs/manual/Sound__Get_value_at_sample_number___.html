Sound: Get value at sample number...

A command available in the Query submenu if you select a Sound object. The Info window will show the amplitude (sound pressure in Pascal) at a specified sample number. If the sample number is less than 1 or greater than the number of samples, the result is undefined.

Setting

Sample number
the sample number at which the value is to be evaluated.
