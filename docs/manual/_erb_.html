erb

A function that can be used in Formulas.

Syntax and semantics

erb (f)
compute the auditory equivalent rectangular bandwidth (ERB) in hertz, for a specified acoustic frequency f (also in hertz).

Definition

erb (f) = 6.23·10−6 f2 + 0.09339 f + 28.52
