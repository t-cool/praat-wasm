Table: Box plots...

A command to draw a box plots for each factor from the data in other column(s) of the selected Table object, where the factors will be sorted in natural sort order.

Example

To draw separate box plots for the male, female and children F0 for the Peterson & Barney (1952) data:

Create formant table (Peterson & Barney 1952)
 Box plots: "F0", "Type", 70, 400
 Text left: "yes", "F0 (Hz)"
