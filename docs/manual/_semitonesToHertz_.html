semitonesToHertz

A function that can be used in Formulas.

Syntax and semantics

semitonesToHertz (x)
convert a value x along a logarithmic acoustic scale in semitones (relative to 100 Hz) to an acoustic frequency in Hz.

Definition

semitonesToHertz (x) = 100 exp (x ln 2 / 12).
