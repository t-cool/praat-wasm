Get nearest index from time...

A query to ask the selected tier object (DurationTier, IntensityTier, PitchTier) which point is nearest to the specified time.

Setting

Time (s)
the time near which you want to get the point index.

Return value

This query returns the index of the point whose time is closest to Time. It is undefined if there are no points.
