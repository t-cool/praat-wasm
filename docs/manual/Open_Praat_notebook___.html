Open Praat notebook...

A command in the Praat menu for editing an existing Praat notebook. It creates a NotebookEditor and asks you to select a file. If you click OK, the file is read into the NotebookEditor window, and you can run and edit it; if you click Cancel, you get an empty notebook, as with New Praat notebook.
