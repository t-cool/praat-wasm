World menu

One of the menus of the Picture window.

Purpose

To draw text, lines, curves or shapes.

The coordinates for commands in the World menu are to be specified as world coordinates, i.e. in the same coordinates as the latest drawing (e.g. seconds horizontally and pascals vertically) or as specified by the latest Axes... command.
