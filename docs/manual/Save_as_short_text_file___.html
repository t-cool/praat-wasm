Save as short text file...

One of the commands in the Save menu.

Availability

You can choose this command after selecting one or more objects.

Behaviour

The Objects window will ask you for a file name. After you click OK, the objects will be written to a text file on disk.

File format

The format is much shorter than the one described at Save as text file.... Most of the comments are gone, and there is normally one piece of data per line.

The file can be read again with the all-purpose Read from file....
