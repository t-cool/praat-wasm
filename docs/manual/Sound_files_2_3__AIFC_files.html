Sound files 2.3. AIFC files

AIFC is short for AIFF(C) or AIFF-C, i.e. the Audio Interchange File Format (§2.2) with optional compression.

Praat reads uncompressed AIFC files with plain 8-bit, 16-bit, 24-bit or 32-bit encoding, but does not support compressed AIFC files.
