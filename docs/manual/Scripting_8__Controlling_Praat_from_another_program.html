Scripting 8. Controlling Praat from another program

Sendpraat is a function for sending messages to a running Praat. It is also a Windows, MacOS, or Linux console program with the same purpose.

As sendpraat cannot start up a new instance of Praat, you may often want to use praat --send instead (see Scripting 6.9. Calling from the command line).
