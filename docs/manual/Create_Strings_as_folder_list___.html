Create Strings as folder list...

A command in the New menu to create a Strings object containing a list of folders in a given parent folder. It works completely analogously to Create Strings as file list....

Script usage

See Create Strings as file list.... If you don’t need the resulting Strings object, it may be easier to use folderNames$# instead.
