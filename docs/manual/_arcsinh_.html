arcsinh

A function that can be used in Formulas.

Syntax and semantics

arcsinh (x)
compute the inverse hyperbolic sine of x.

Definition

arcsinh (x) = ln (x + √(1+x2))

Tests

assert arcsinh (-1.0) < 0
 assert arcsinh (0.0) = 0
 assert arcsinh (1.0) > 0
 assert arcsinh (undefined) = undefined
