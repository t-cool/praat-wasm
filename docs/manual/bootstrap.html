bootstrap

The bootstrap data set is a random sample of size n drawn with replacement from the sample (x1,...xn). This means that the bootstrap data set consists of members of the original data set, some appearing zero times, some appearing once, some appearing twice, etc.

More information can be found in Efron & Tibshirani (1993).
