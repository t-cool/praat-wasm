Phonetic symbols

To draw phonetic symbols in the Picture window or in the TextGridEditor, make sure that you have installed the Charis and/or Doulos SIL font, for instance from sil.org or from praat.org. You can either type the symbols directly (if your computer has an input method for them), or use backslash trigraphs as described in:

• Phonetic symbols: consonants
• Phonetic symbols: vowels
• Phonetic symbols: diacritics
