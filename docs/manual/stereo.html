stereo

A stereo sound is a sound consisting of a left and a right channel. Praat can read and write both mono and stereo sounds.

Reading a stereo sound from disk

If you read a Sound into Praat from disk with Read from file..., and that sound file is a stereo file, the resulting Sound object will be a stereo Sound.

If you want to look at the left or right channel separately, you can use Extract one channel....

Saving a stereo sound to disk

If you choose Save as WAV file..., and the sound is a stereo sound, then the sound file will be a stereo file. If you have two or more mono sounds and you want to combine them into one stereo sound, you can use Sounds: Combine to stereo.
