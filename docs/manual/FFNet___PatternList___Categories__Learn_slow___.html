FFNet & PatternList & Categories: Learn slow...

To learn an association you have to select a FFNet, a PatternList and a Categories object.

Preconditions

The number of columns in a PatternList must equal the number of input units of FFNet.

 Algorithm

Steepest descent

Preconditions

The number of rows in a PatternList must equal the number of categories in a Categories.
The number of unique categories in a Categories must equal the number of output units in FFNet.
