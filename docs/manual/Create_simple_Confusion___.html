Create simple Confusion...

Creates a square confusion matrix with equal stimulus labels and response labels.

Example

The command Create simple Confusion: "simple", "u i a" results in the following Confusion:

u i a ! The response labels
 u 0 0 0 ! Responses on stimulus u,
 i 0 0 0 ! Responses on stimulus i
 a 0 0 0 ! Responses on stimulus a
