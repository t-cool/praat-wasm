hertzToBark

A function that can be used in Formulas.

Syntax and semantics

hertzToBark (x)
convert from an acoustic frequency x in Hz to an auditory Bark-rate (auditory spectral frequency; place on basilar membrane).

Definition

hertzToBark (x) = 7 ln (x/650 + √(1 + (x/650)2))
