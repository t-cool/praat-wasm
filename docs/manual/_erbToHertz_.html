erbToHertz

A function that can be used in Formulas.

Syntax and semantics

erbToHertz (x)
convert from an auditory ERB-rate x to an acoustic frequency in Hz.

Definition

erbToHertz (x) (14680 d - 312) / (1 - d), where d = exp ((x - 43) / 11.17)
