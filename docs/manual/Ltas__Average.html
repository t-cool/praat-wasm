Ltas: Average

The command Average becomes available under Combine when you select one or more Ltas objects.

When you choose Average, a new Ltas called “averaged” will appear in the list of objects. This new Ltas will contain the average of the Ltases that you had selected.

Algorithm

The averaging takes place in the energy domain. For instance, if you select three Ltases, the values in the three Ltases will be converted from dB to Pa2, then added up, then divided by 3, then converted from Pa2 to dB.
