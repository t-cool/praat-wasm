abs##

A function that can be used in Formulas.

Syntax and semantics

abs## (m##)
compute the absolute value (abs) of each cell of the matrix m##.

Examples

a## = abs## ({{ -1, 5.5, 0, -456.789 }, { 34, -45, 18.0, 0 },
 ... { 1e-89, -3.2e-307, 6e200, -2345e-2 }})
