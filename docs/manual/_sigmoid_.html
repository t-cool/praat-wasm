sigmoid

A function that can be used in Formulas.

Syntax and semantics

sigmoid (x)
compute 1 / (1 + e-x) or 1 − 1 / (1 + ex)

Domain and range

sigmoid (x) is defined for all real numbers x. The range are real values between 0 (not included) and 1 (not included).
