Create NoCoda grammar

A command in the Constraint grammars submenu of the New menu, for creating an OTGrammar object with a NOCODA grammar.

The grammar has a structural constraint NOCODA, and a faithfulness constraint PARSE. There are only two underlying forms: “pa” and “pat”, where the latter can have “pa” and “pat” as surface forms.

For a examples using this grammar, see OT learning 2.1. Viewing a grammar.
