Read separate channels from sound file...

A command in the Open menu of the Objects window. You use this if you want to get the channels of a stereo (or multi-channel) sound file as two (or more) separate Sound objects in the list. If the file name is hello.wav, Praat will name the channels hello_ch1, hello_ch2, and so on.
