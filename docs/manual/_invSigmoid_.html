invSigmoid

A function that can be used in Formulas.

Syntax and semantics

invSigmoid (x)
compute the inverse sigmoid.

Definition

invSigmoid (x) = ln (x / (1 − x)).

Domain and range

invSigmoid (x) is defined for 0 < x < 1. The possible results are all real numbers.
