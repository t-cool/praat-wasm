Sound: Get time of maximum...

A command available in the Query submenu if you select a Sound object. The Info window will show the time (in seconds) associated with the maximum pressure in a specified time range.

Settings

Time range (s)
the time range (t1, t2). Values outside this range are ignored, except for purposes of interpolation. If t1 is not less than t2, the entire time domain of the sound is considered.
Interpolation
the interpolation method (none, parabolic, cubic, sinc70, sinc700) of the vector peak interpolation. The standard is “sinc70” because a Sound object is normally a sampled band-limited signal, which can be seen as a sum of sinc functions.
