This picture illustrates the following properties of the sine function:

• sin (x) is zero for all values of x that are integer multiples of π.
• The extrema of sin (x) are −1 and +1.
• sin (x) is −1 or +1 wherever x is an odd multiple of π/2.
• The function is periodic with period 2π, i.e., sin (x+2π) = sin (x).
• The function is symmetric around π/2, i.e. sin (π/2−x) = sin (x).
• The function is antisymmetric around 0, i.e. sin (−x) = − sin (x); in other words, if you take the sine curve and rotate it by 180 degrees around the origin (i.e. the point (0, 0)), then you get the same curve again.
